## Deskripsi Tugas

Tugas ini meminta mahasiswa untuk membuat program JavaScript yang dapat menampilkan tanggal hari ini dalam format Bahasa Indonesia. Contoh format keluaran yang diharapkan adalah sebagai berikut:

```
Sabtu, 18 April 2026
```

Nilai tanggal tidak harus sama dengan contoh, namun susunan formatnya harus mengikuti pola tersebut. Untuk memenuhi kebutuhan ini, digunakan objek `Intl.DateTimeFormat` yang sudah tersedia secara bawaan di JavaScript, sehingga penyesuaian locale dapat dilakukan tanpa perlu menyusun string tanggal secara manual.

---

## Ketentuan

Program harus memenuhi ketentuan berikut:

- Menggunakan `Intl.DateTimeFormat`
- Menggunakan locale Bahasa Indonesia (`id-ID`)
- Menampilkan komponen tanggal berikut:
  - Nama hari (weekday)
  - Tanggal (day)
  - Nama bulan (month)
  - Tahun (year)
- Susunan format akhir: `Hari, Tanggal Bulan Tahun`

---

## Kode Program

```javascript
const now = new Date();

const parts = new Intl.DateTimeFormat("id-ID", {
    weekday: "long",
    day: "numeric",
    month: "long",
    year: "numeric"
}).formatToParts(now);

const result = `${parts[0].value}, ${parts[2].value} ${parts[4].value} ${parts[6].value}`;

console.log(result);
```

## Penjelasan Program

Program dimulai dengan membuat objek `Date` untuk mendapatkan waktu saat ini. Selanjutnya, `Intl.DateTimeFormat` dikonfigurasi menggunakan locale `"id-ID"` agar nama hari dan bulan yang dihasilkan menggunakan Bahasa Indonesia.

Metode `formatToParts()` dipilih karena mengembalikan setiap komponen tanggal sebagai elemen terpisah dalam bentuk array objek. Dengan demikian, urutan dan tanda pemisah antar komponen dapat diatur secara eksplisit — dalam hal ini, tanda koma disisipkan setelah nama hari agar sesuai format yang diminta.

Pendekatan ini lebih andal dibandingkan menyusun string tanggal secara manual, karena mengikuti standar internasionalisasi JavaScript dan hasilnya konsisten tanpa bergantung pada asumsi format tertentu.
